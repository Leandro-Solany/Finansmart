## ? Quick Start

```bash
# Clone o repositório
git clone https://github.com/Leandro-Solany/Finansmart.git

# Entre no diretório
cd Finansmart

# Restaure as dependências
dotnet restore

# Execute a aplicação
dotnet run
```

Acesse: http://localhost:5000

## ?? Docker

```bash
docker-compose up -d
```
